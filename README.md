# Volumes

_A 3D bookshelf planning tool built with Three.js and TypeScript. Users model their physical shelves, enter their book collection, and get a 3D
visualization of how all their books fit with multiple arrangement algorithms (by volume, by series, by decoration, etc...)._

## Tech Stack

- [Three.js](https://threejs.org/) — 3D rendering
- [Vite](https://vitejs.dev/) — build tool
- [TypeScript](https://www.typescriptlang.org/) — strict mode
- [Biome](https://biomejs.dev/) — linting and formatting
- [pnpm](https://pnpm.io/) — package manager
